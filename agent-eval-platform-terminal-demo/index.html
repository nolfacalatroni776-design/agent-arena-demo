平台架构方案 ?

这不是单页 demo 架构，而是可演进到私有化部署、在线评估和数据资产市场的终态架构。

Execution and evaluation decoupled

ADR-1

执行与评估两个独立 pipeline，以轨迹 + 终态快照为中间产物。

ADR-4

沙箱用强隔离运行时，Checker 与 Agent 容器物理分离。

ADR-7

所有分数相关配置采用不可变版本快照。
