需求分析文档

Agent Arena · 由 Appen 数据标注平台拓展的 Agent 竞技 / 评测 / 训练数据闭环产品

需求分析 需求设计 技术方案 风险点

1. 背景与战略目标

Appen 现有数据标注平台已沉淀大量场景化专家、真实业务难例与多模态精标数据。在 LLM Agent 应用爆发的当下，企业客户的痛点已从"我要训练数据"升级为：

  • "我的 Agent 在我的真实业务场景里到底行不行？"
  • "它哪里不行？需要哪类数据 / 微调 / 工具补救？"
  • "市面上同场景哪个 Agent 是 SOTA？我能不能直接用？"

Agent Arena 把 Appen 的难例 → 评测 → 数据 → 训练 → 再评测形成商业闭环，将平台从"卖数据"升级为"卖能力诊断 + 卖数据 + 卖最佳 Agent 方案"。

北极星指标：每季度产生 N 份付费的 Agent Insight Report → 转化为训练数据采购 GMV。

4. 核心场景与功能需求

4.1 客户接入与 Agent 部署

  • 支持 3 种接入：Docker 镜像 / HTTP API（含 OpenAI 兼容） / 在线 IDE 构建。
  • 沙盒拉起、健康检查、smoke test、资源规格选择、MCP 工具白名单。
  • 多版本管理：每个 Agent 可保留 N 个版本，竞技场以版本号锚定。

4.2 竞技场 (Arena) & 难例 PK

  • 竞技场按"业务场景 × 任务集"组织，每个 Arena 有：奖金池、报名期、对决期、复盘期。
  • 难例任务集 = 平台精标 + 专家众包提交（带审核工作流）。
  • 双盲评测：评分 = 0.55 × 正确性 + 0.20 × 安全合规 + 0.15 × 工具调用质量 + 0.10 × 成本效率。
  • 实时 leaderboard（30s 刷新）、对决回放、可视化推理轨迹。

4.3 难例众包（专家有奖出题）

  • 提交：标题、场景、Rubric、附件、自评难度、期望奖励。
  • 审核：Appen 专家 + 自动校验（重复率、可解性、Rubric 可执行）。
  • 分润：基础奖励 + 被竞技场采纳的 20% 加成 + 被采购数据集采纳的版税。

4.4 Agent 问题分析报告（核心商业入口）

  • 多 Arena 聚合 → 雷达图（6 维能力） + 错误归因 TOP N + 失败样本下载。
  • 每类错误绑定具体修复建议 → 推荐 Appen 数据集 / 微调服务 / 工具 MCP。
  • 报告可一键下单"训练数据集 + 微调服务 + 复测"打包。

4.5 训练数据集商城

  • 按场景 × 错误类型上架；提供样本预览、License、月增量、覆盖度。
  • 支持订阅式（持续更新）和买断式。
  • 定制采集咨询入口（接 Appen 现有交付链路）。

4.6 Agent 方案推荐 / 市场

  • 基于竞技场战绩自动评出场景 SOTA 与"高性价比之选"。
  • 对外提供：API 订阅、私有部署询价、与 Appen 数据捆绑销售。

5. 用户旅程 (User Journey)

客户登录 → 部署/构建 Agent → 选择目标 Arena → 沙盒预检 → 报名 → 对决 → 看 leaderboard
                                                              ↓
                                           查看 PK 回放 → 生成「Agent 问题分析报告」
                                                              ↓
                                        采购推荐数据集 / 微调服务 → 复测 → 排名提升
                                                              ↓
                                              （若仍不达标）订阅市场上的 SOTA Agent

6. 非功能需求

类别要求
性能对决期单 Arena 支持 200 战队并发；评测吞吐 ≥ 50 任务/秒/Arena；leaderboard P95 < 800ms。
沙盒安全每个 Agent 隔离命名空间，禁止跨账号网络访问；外网调用走代理 + 审计。
合规训练数据需具备授权链；医疗 / 金融场景需脱敏审计 + 行业资质。
可观测所有推理轨迹保留 90 天可重放；失败样本可一键导出训练对。
多语UI 中英双语；难例自带语言标签。

7. 范围边界（Out of scope · MVP 不做）

  • 不做开源模型在线训练（仅做评测 + 数据采购 + 微调服务对接）。
  • 不做通用 LLM 排行榜（仅做"场景 × Agent"排行）。
  • 不做支付级账户体系（MVP 用平台余额 + 线下结算）。

8. 成功度量 (KPI)

  • 季度内活跃 Arena 数 ≥ 12；专家提交难例 ≥ 800 条 / 季度。
  • 付费 Agent Insight Report ≥ 60 份 / 季度，报告 → 数据采购转化率 ≥ 35%。
  • 数据集商城 GMV 同比 +50%。
  • NPS（客户 Agent 团队）≥ 45。
